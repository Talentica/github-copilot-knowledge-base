## To ignore env, secrets or other files from Copilot in VScode:
.vscode/settings.json add below config.

```
  "files.associations": {
    ".env*": "dotenv",
    ".config*": "config"
  },
  "github.copilot.enable": {
    "*": true,
    "config": false
    "dotenv": false
  }
```
