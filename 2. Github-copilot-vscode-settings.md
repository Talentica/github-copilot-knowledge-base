## Individual - To ignore secrets or other files from copilot use below settings:
.vscode/settings.json add below config.

```
  "files.associations": {
    ".env*": "dotenv",
    ".config*": "config"
  },
  "github.copilot.enable": {
    "*": true,
    "config": false
    "dotenv": false
  }
```
